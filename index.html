Welcome !!!

About

  • Visualizing the state of an OpenFlow network is now easier with Panorama.
  • Panorama sets up instantaneously, requires nothing but Python.
  • Panorama is a lightweight application designed to ease the work of OpenFlow users, explorers & researchers.

Features

  • Panorama collects and depicts vital information about the network such as:
    1. N/w Configuration,
    2. Port Stats,
    3. Aggregate Stats,
    4. Flow Stats,
    5. Link Utilization.
  • Real-time information about the network, all at one place.
  • Toggle through tabs and enjoy hassle free Panoramic view of the network.
